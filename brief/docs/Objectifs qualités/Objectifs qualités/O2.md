# O2 - Facilité d'apprentissage pour un naturaliste

Un possesseur de PR sans formation informatique doit pouvoir réaliser le **cycle complet** (importer une nuit → consulter les observations → valider quelques évènements → exporter le CSV) après une **démonstration de quinze minutes** par un pair.

- **Catégorie** : [Convivialité](../Catégorie%20de%20qualité/Convivialité.md)
- **Critère** : [Facilité d'apprentissage](../Critère%20de%20qualité/Facilité%20d’apprentissage.md)
- **Scénario associé** : [SC1](../Scénario/SC1.md)

## Justification métier

Le réseau VigieChiro repose sur le bénévolat. Une application qui exigerait plusieurs heures de formation perdrait une majorité de ses utilisateurs en cours de route.

## Mesure

- Test utilisateur informel : une personne qui découvre l'application parvient à réaliser le cycle complet sans aide après une démonstration de 15 min.
- Vocabulaire de l'IHM cohérent avec le langage métier (ne pas écrire « parser le fichier » mais « importer une session »).
