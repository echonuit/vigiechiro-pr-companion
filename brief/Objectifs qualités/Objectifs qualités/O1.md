# O1 - Portabilité multi-OS

L'application doit fonctionner de manière identique sur les principaux systèmes d'exploitation utilisés par les naturalistes (**Windows 10/11, Linux Ubuntu/Debian récents, macOS récent**) sans nécessiter d'installation système lourde.

- **Catégorie** : [Portabilité](../Catégorie%20de%20qualité/Portabilité.md)
- **Critère** : [Facilité d'installation](../Critère%20de%20qualité/Facilité%20d’installation.md)

## Justification métier

La communauté VigieChiro est très hétérogène techniquement : on y croise aussi bien des chercheurs sous Linux que des bénévoles sous Windows ou des biologistes sous macOS. Imposer un OS reviendrait à exclure une partie du public.

## Mesure

- Le `./mvnw javafx:run` produit le même comportement sur les trois OS cibles dans la CI (matrice GitHub Actions).
- Aucune dépendance native binaire spécifique à un OS dans le `pom.xml` (en dehors de JavaFX, qui gère son propre cross-platform).
