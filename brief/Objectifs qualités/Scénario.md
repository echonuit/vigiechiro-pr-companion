# Scénarios

Les scénarios donnent corps aux objectifs qualités les plus exigeants en les ancrant dans une situation d'usage concrète.

| Id | Description | Objectif lié |
|---|---|---|
| [SC1](Scénario/SC1.md) | Un possesseur de PR qui découvre l'application réussit à traiter sa première nuit après quelques explications simples (*onboarding*). | [O2](Objectifs%20qualités/O2.md) |
| [SC2](Scénario/SC2.md) | Aucun identifiant observateur, identifiant de participation ou numéro de PR ne fuit hors de l'environnement local de l'utilisateur (*privacy*). | [O8](Objectifs%20qualités/O8.md) |
| [SC3](Scénario/SC3.md) | Dans une version ultérieure, l'application accueille un nouveau protocole VigieChiro (Pédestre, Routier...) sans réécrire son cœur (*extendability*). | [O6](Objectifs%20qualités/O6.md) |
| [SC4](Scénario/SC4.md) | Avec un fichier de traduction approprié, tous les textes affichés et imprimés apparaissent dans une langue tierce (*accessibility / i18n*). | |
