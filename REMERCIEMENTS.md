# 🙌 Remerciements

Le projet **VigieChiro PR Companion** a été construit dans le cadre de la **SAÉ 2.01** du **BUT Informatique** de l'**IUT d'Aix-Marseille** (promo 2026).

Il est le fruit du travail de **21 équipes** (**92 étudiantes et étudiants**) qui ont contribué au fil du semestre : conception, développement de l'interface, tests, revues de code et gestion de projet.

Chaque équipe a travaillé dans son propre dépôt au sein de l'organisation [IUTInfoAix-S201-2026](https://github.com/IUTInfoAix-S201-2026). Les pseudonymes ci-dessous renvoient aux profils GitHub publics, et les liens d'équipe vers les dépôts de travail correspondants.

> Les équipes et leurs membres sont présentés par ordre alphabétique : ce document met en valeur une contribution collective, il n'établit aucun classement.
