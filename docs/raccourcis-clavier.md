# Raccourcis clavier

VigieChiro se pilote aussi bien au clavier qu'à la souris. Cette page récapitule les raccourcis
disponibles : ceux qui marchent partout, puis ceux propres à l'écran de qualification.

## Partout dans l'application

| Raccourci | Action |
|---|---|
| <kbd>Alt</kbd> + <kbd>←</kbd> | Revenir à l'écran précédent |
| <kbd>Alt</kbd> + <kbd>Début</kbd> | Revenir directement à l'accueil |
| <kbd>Tab</kbd> | Parcourir les éléments au clavier (cartes, champs, boutons) |
| <kbd>Échap</kbd> | Fermer la fenêtre ouverte (modale) |

Sur les écrans organisés en cartes (l'accueil, « Mes sites »), <kbd>Entrée</kbd> ou <kbd>Espace</kbd>
ouvre la carte qui a le focus.

!!! tip "Navigation au clavier"
    La touche <kbd>Tab</kbd> déplace le focus d'un élément à l'autre, avec un liseré visible sur
    l'élément actif. <kbd>Maj</kbd> + <kbd>Tab</kbd> revient en arrière.

## Écran Qualification (écoute et verdict)

Cet écran est pensé pour un travail rapide au clavier, sans lâcher l'écoute : on écoute, on juge, on
enregistre, on passe au suivant.

| Raccourci | Action |
|---|---|
| <kbd>Espace</kbd> | Lecture / pause de l'extrait |
| <kbd>O</kbd> | Verdict : OK |
| <kbd>D</kbd> | Verdict : Douteux |
| <kbd>J</kbd> | Verdict : À jeter |
| <kbd>Entrée</kbd> | Enregistrer le verdict |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Passer d'une séquence à l'autre, lorsque la **liste des séquences** a le focus |

!!! note "Pendant la saisie d'un commentaire"
    Les touches de verdict (<kbd>O</kbd>, <kbd>D</kbd>, <kbd>J</kbd>) sont suspendues tant que le
    curseur est dans un champ de texte, pour ne pas gêner votre frappe.
